Q.1 - Difference between flex row and flex column.

Ans. - The CSS FlexBox is a one dimensional layout. It is helpful in allocating and aligning the space among items in a container. It is done using display:flex property of CSS.

Flex row - flex-direction:row; It is a default value and is used to display items horizontally, as a row on a webpage.

Flex column - flex-direction:column; property is used to display items vertically, as a column on a webpage.

Q.2 - Explain Inline, Internal and External Stylesheet.

Ans. - There are three types of CSS - inline, internal and external that can be added to html documents.

Inline - by using 'style attribute' inside opening tag of any element.

Internal - by using a 'style tag' inside head tag.

External - by using a 'link tag' inside head tag to link to an external CSS file.

Q.3 - Justify-content allows you to do what?

Ans. - It is used to align the flexible box container's item along the main axis of a flex container. This property manages space distribution between and around content items in a flex container.

It has following values :

flex-start : align flex items at the start of the container.

flex-end : align flex items at the end of the container.

center : align flex items at the center of the container.

space-between : align items evenly with the first item at the start and last item at the end.

space-around : align items evenly with the equal space around them.

space-evenly : align items with the equal space between them.

Q.4 - Difference between absolute and relative positioning.

Ans. - Relative Positioning - 'position:relative;' position an element relative to its normal position in the document flow. When we apply the top, right, bottom, left properties, the element shifts from its original position.However, the space it originally occupy remains reserved in the document.

Absolute positioning - 'position:absolute;' position element relative to its nearest positioned ancestor, or to the initial containing block if there is no positioned ancestor. Absolute positioned elements are removed from the normal document flow and can overlap other elements.

Q.5 - What is the z-index in CSS?

Ans. - CSS z-index property is used to control the stacking order of overlapping elements, which decides whether an element appears on top or behind others based on their assigned value. Each element's position along the imaginary z-axis is expressed as an interger(positive, negative, or zero).

Q.6 - What is the difference between padding and margin?

Ans. - Margin - Margin provides space outside an element, creating distance between it and other elements or the page's edges.

Padding - Padding provides space inside an element separating its content from its border.
